# Alert

Alert is an inline component used for displaying different types of messages.

## Import

```js
import { Alert } from '@fabric-ds/react';
```

## Visual Options

### Negative

```jsx example
<Alert type="negative">
  This is the message text that can be short or a little bit long
</Alert>
```

### Positive

```jsx example
<Alert type="positive">
  This is the message text that can be short or a little bit long
</Alert>
```

### Warning

```jsx example
<Alert type="warning">
  This is the message text that can be short or a little bit long
</Alert>
```

### Info

```jsx example
<Alert type="info">
  This is the message text that can be short or a little bit long
</Alert>
```

### Neutral

```jsx example
<Alert type="neutral">
  This is the message text that can be short or a little bit long
</Alert>
```

### Other examples

```jsx example
<Alert type="info">
  <p className="font-bold">Title Caption-Strong</p>
  <p>This is the message text that can be short or a little bit long</p>
</Alert>
```
